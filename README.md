# 📈 Data Analyst Forex Trading Signal AI Agent

## Project Overview

This project was developed as part of **Project 1C** under the **Zetheta Algorithms Private Limited WorkBridge Program**.

The objective of this project is to analyze historical Forex market data, engineer technical indicators, build a machine learning model for trading signal analysis, and visualize insights through interactive Power BI dashboards.

The project demonstrates an end-to-end data analytics workflow, including data preprocessing, feature engineering, machine learning, and business intelligence reporting.
